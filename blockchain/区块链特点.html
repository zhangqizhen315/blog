# 区块链特点

一般认为区块链的特有:

  • 分布式
  • 去中心化
  • 去信任
  • 不可篡改
  • 可编程

# 对区块链的理解

# 可编程

  • 强大的脚本功能和可扩展性
  • 通过特定的算法来计算权益, 信用与身份的真伪, 算法以强大的加密技术, 根据不同场景, 灵活编定不同的智能合约.

# 信用协议 ( 智能合给 )

  • 交易不可逆, 数据不可篡改. 将过去对威权第三方的信任, 转化为对算法对数学的信任, 对交易行为本身交易对象的信任.
  • 信任的类型
    • 交易行为的诚信
    • 交易对象的诚信
  • 信任的方法
    • 工作量证明等共识机制
    • 信用真实, 时间戳

# 价值互联网

  • 以处理竞争性资源的方式来处理非竞争性资源, 实现了价值互联
  • 万维网是信息互联网, 区块链实现了价值互联

# 共享账簿

  • 侧重于区块链作为分布式记账的一面
  • 只是区块链在数据层的创新, 这种提法忽略了在建立信用互联网协议的创新.
