# 极简古希腊哲学史

如果我看到朋友圈或公众号里有一篇文章叫 《极简古希腊哲学史》 我一定打开都不打开.

然而必竟事出有因.

周二早上, 我在上班的地铁上, 收到教友奥斯同学的微信, 让我带他学习古希腊哲学.

自然非常惭愧. 我这票友水平, 怕是不能引人登堂入室. 开始想着, 推荐几本自己觉得靠谱的书. 但是想来, 这些书要是想看就能看得进去, 他大概早就看.

就我的感受, 古希腊哲学都是贴着常识走的. 真正拦着人亲近的, 是对于现代人来说, 实在没有了解它们的趣味. 从现代理性来看, 那些想法都是被科学证实的缪误罢了.

于是我就产生了写这篇极简史的想法, 一是想说说对我来说, 古希腊哲学的趣味所在；其次也是想用最少的文字来给出古希腊哲学的框架 -- 对我来说, 想提出有价值的诠释有困难, 但给出谁说了啥的基本事实还是没大问题的.

# 古腊希哲学的趣味所在

# 元思维

两年前, 刚开始写公众号的时候, 写过一个篇讲古希腊哲学趣味的文章, 就说的这事.

作为受过现代科学教育的人看古希腊哲学, 常会觉得当时所谓哲思都是对世界的胡乱的猜测罢了.

这也难免. 那是个连三段论这样基本的逻辑工具都还没诞生的年代.

然而正是这个没有理性工具的年代, 奠定了人类理性的基础 -- 至少是西方理性罢. 不但如此, 我们现在所依赖的理性工具都是在那个时代, 被当时的哲人们所建立起来的.

用现在流行的话说, 哪个时代的思想, 是我们现在思想的元思想.

探寻建构人类理性的底层代码, 是了解古希腊哲学的第一个趣味.

# 大变革与根基性

哲学繁荣的时代往往是大变革时代.

古希腊哲学的时代被称为轴心时代. 同一时期, 古印度产生了沙门思潮, 中国正在经历百家争鸣.

东西方在同一时期奠定文化基调并非偶然, 这时的人类正在经历一场大变革.

这场变革是人类从渔猎文明向农耕文明的转变.

渔猎的生存方式是即时反馈的. 德性与运气的结果能快速而明显的反映出来. 因此人们相信神人共在, 尊崇勇气与荣誉. 人们无需作长远的打算, 渔猎生活的基调是快乐和无忧无虑的.

农耕的生存方式却是异步的, 要整整劳作一年, 才能等来一次收获. 勇气与荣誉被规划和劳作所替代, 而在平淡的日常中, 人们也很难直观的感受到神灵的存在.

农耕生活的基调是痛苦和忍耐.

这种转变体现在荷马史诗和赫西俄德的诗歌中. 赫西俄德不再像荷马一样歌颂英雄的德性, 而是向 "只知吃喝不知羞耻的牧人" 教谕如何安排工作与时日.

传说, 荷马与赫西俄德比赛, 赫西俄德获得了桂冠. 这自然不是史实, 但却是那场大变革的写照.

在这样一场大变革中, 人们必须重新思考世界是什么, 人是什么, 人与世界的关系是什么. 这就是古希腊自然哲学最初讨论的内容.

所有古希腊哲学的流派都脱胎于神话和宗教, 而最终走向了对神话和宗教传统的反叛, 这就是智者学派.

智者学派提出 "人是万物的尺度", 看似树立了人在世界的位置, 但其实却让人失去了一切根基性, 智慧变成了 "使弱的道理战胜强的道理" 的技术.

苏格拉底, 柏拉图, 亚里士多德三圣哲对自然哲学和智者学派进行反思, 重新为人建立了与世界的联系, 找回了人的根基性.

# 特殊性塑造世界

关于李约瑟问题, 陈嘉映老师说, 其实我们不应该问为什么中国没有发生工业革命. 因此这个世界上大多数国家都没有发生工业革命. 我们应该问的是, 为什么工业革命发生在英国.

在这件事上, 与我们想象的 "天下大势, 浩浩荡荡" 不同, 历史不是由多数人的普遍性决定的, 而是由少数人的特殊性决定的.

为整人类理性奠基的, 是非常在当时非常与众不同的希腊人.

希罗多德在他的《历史》中讲过一个故事, 萨第斯的僭主坎道列斯为了虚荣让他的近侍偷看自己的妃子, 而这个妃子的复仇使得亚底亚政权易手. 这讲这个故事时, 希罗多德特意强调了一句: 这个世界上所有国家的人都觉得了被人看到祼体是奇耻大辱, 除了希腊人.

我们现在觉得古希腊人光着腚举行运动会是件很怪的事情, 当时其他民族也是这样觉得的.

在当时, 希腊民族不是一个具有代表性的民族, 甚至不是一个让其他民族觉得有什么重要性的民族. 在波斯人崛起前, 他们被吕底亚人征服. 而在波斯人打败吕底亚人后, 也没怎么把他们放在眼里, 而只是派了个总督去接收吕底亚的人遗产.

而雅典人则又是希腊民族中的异类. 雅典人在一直在强调自己的民主制, 这不正说明, 当时其他的城邦并不如此么?

正是这样一群异类中异类, 奠定了人类理性的基础. 因此, 我们是不是可以象问为什么工业革命发生在英国一样, 来问, 为什么人类理性精神起源于古希腊.

# 极简史

任何极简的知识都是可疑的. 我可以负责任的说, 7 天精通不了 JAVA, 连入门都不行.

因此, 所谓 "极简史" 并不是史, 只是一个最粗的轮廓.

# 简单的四分法

最简的古希腊史分期是: 前苏格拉底哲学和希腊古典哲学.

希腊古典哲学特指苏格拉底, 柏拉图, 亚里士多德师徒三人的哲学. 他们之前的哲学通常统称为前苏格拉底哲学.

而在我这里, 因为对巴门尼德哲学特有的情结, 把前苏格拉底哲学分为三段: 巴门尼德之前的哲学家, 巴门尼德的哲学, 巴门尼德之后的哲学.

# 巴门尼德之前的哲学

巴门尼德之前的哲学有三派: 伊奥尼亚哲学, 南意大利哲学和赫拉克利特哲学.

如果试着用最简单的话来说, 伊奥尼亚哲学认为世界是混沌的, 南意大利哲学认为世界是秩序的, 而赫拉克利特哲学是他们的折衷.

# 伊奧尼亚哲学

古希腊第一个哲学家的米利都的泰勒斯, 他说: 万物是水. 水在古代哲学中有着特殊的地位, 不独希腊如此, 中国亦然. 我觉得这和母系时代的女神崇拜有关.

但无论如何, 泰勒斯的是一个哲学家, 他对于 "万物是水" 有一套理性表述: 水是始基. 所谓始基, 是指万物由它而来又复归于它.

泰勒斯的学生阿那克西曼德解释他老师的学说, 如果有一个东西, 万物都由它而来又复归于它, 那它一定没有固定的型态. 因为有固定型态的东西, 无法转化成其他东西. 只有 "无定型" 的东西才能是始基.

阿那克西曼德的学生阿那克西美尼认为, 水还不够不定型, 比水不定型的是气. 因此他认为始基是气.

# 南意大利哲学

南意大利哲学就是毕达哥拉斯哲学. 与伊奥尼亚哲学认为世界是在不同世界间转化不同, 他们认为世界存在于一种 永恒的秩序 中. 这个秩序就是数.

南意大利哲学认为 万物皆数. 数是世界的无素.

与我们现在对于数的认知不同, 他们认为数是个东西, 是实存之物. 这些每一个实存的数都是一个点, 因此世界由数构成, 是在说世界是 "点积" 的.

正是因此, 他们无法想象无理数的存在.

南意大利哲学和伊奥尼亚哲学形成了古希腊哲学的基本张力, 伊奥尼亚哲学认为世界是 混沌的, 流变的, 连续的, 而南意大利哲学认为世界是 秩序的, 稳定的, 点积的.

# 赫拉克利特

赫拉克利特认为世界不是东西, 而是个过程. 这个过程他概括为 "活火有度".

他说世界是一团不断燃烧, 不断熄灭, 又不断再次燃烧的火焰. 是不是听起来有点象黑魂的设定.

世界之所以是一团火焰, 是因为相反者相成. 世间一切者是 通过其对立面获得规定性的. 这种通过对立面获得规定性的过程被他表述成 战争 (争斗).

而正是因为万物通过其对立面获得了规定性, 因此万物又都是 有尺度 的.

# 巴门尼德哲学

在我看来, 巴门尼德哲学古希腊哲学的转折点.

巴门尼德之先, 大家都是在用自己的视角来看待世界. 而巴门尼德之后, 所有人都在回应巴门尼德提出的问题.

在某种意味上, 希腊古典哲学就是在回应巴门尼德的逼问中产生的.

# 一神宗教

我们现在看古希腊神话, 会觉得 "贵圈真乱". 当时人也这么觉.

爱利亚人克塞诺芬尼不满于希腊诸神混乱的伦理关系, 提出了自己的 一神论.

他认为 并不是神话中的诸神创造了人, 而是人按照自己的形象创造了神话中的诸神, 并让他们穿着人的衣服, 说人的语言.

如果神是世界的主宰, 是至善的, 那神必须是唯一的. 因此他提出了一神宗教.

# 巴门尼德

克塞诺诺芬尼并没有给自己的宗教主张提出论证 ( 或者说, 现在并没有他的论证存世 ), 他的学生巴门尼德则对老师的观点提供了哲学基础.

他的观点通常被翻译为: 只有存在的东西存在, 不存在的不存在.

这是一句必然正确的论断, 因为这是同语反复. 就好比说, 只有张启震是张启震, 张启震之外的任何人都不是张启震. 这是所逻辑的基础 "同一律".

从这个必然正确的论断出来, 巴门尼德推出了不一般的结论: 运动不存在, 变化不存在, 部分与整体不存在. 如果世界存在, 那么 万物是一, 没有运动, 没有变化, 没有整体和部分的分别, 世界象是一个 铁板一块的球.

巴门尼德从形式上完全正确的逻辑推出反常识的结论. 但当时其他流派的哲学家完全没有能力从理论上对他的观点进行反驳, 只能在广场是踢石头: 你不是说运动不存在么, 怎么石头飞了?

巴门尼德怎样处理自己理论与常识的割裂呢? 他区分真理与意见. 他说只有神才认识真理, 而人只能认识意见. 因此意见需要符合常识, 而真理则不需要.

他的这个解决方案当然不是一个究竟的方案. 理性与现象相矛盾的问题就此埋下, 真到希腊古典哲学的三圣哲来拯救现象.

# 芝诺

芝诺是巴门尼德的学生. 为了论证老师关于运动不存在的观点, 他提出了四个悖论: 两分法悖论, 阿基里斯追乌龟悖论, 飞矢不动悖论, 游行队伍悖论.

因为我们是 "极简史", 我们就挑比较好说的来简单聊一下.

我们射一支箭出去, 箭飞达终点的过程是无数个瞬间的集合. 而在每个瞬间, 箭必在某一个具体的位置静止着. 即然在这个过程中的每一个瞬箭都是不动的, 所以在整个过程中, 箭也是不动的. 这就是所谓 "飞矢不动" 的悖论.

我们可以把这个悖论与之前古希腊哲学家的观点进行联系: 世界是连续和变化的, 是符合理性和常识的, 如射出去的箭是动的; 世界是点积和静止的, 也是符合理性和常识的， 如在箭飞的每个瞬间, 箭是静止的; 但是综合两个都对的观点, 得到却是不合常识的悖论.

# 巴门尼德之后的哲学

巴门尼德之后的哲学家, 必须要面对巴门尼德的指出的同一律问题.

同一律使 "始基论" 破产了. 始基论意味着世界必须不断转化, 而巴门尼德已经论证了, 转化是不可能的.

所以巴门尼德之后的哲学家必须要找到另一个范畴来替代 "始基", 同时需要找到一种机制来替代 "转化".

# 恩培多克勒

即然单一始基的转化不能解释世界. 那就用个元素的组合来解释.

这就是恩培多克勒解决方案: 世界由水火土气四种元素组合而成. 为了解释四处元素结合的动力, 他又提出了两个爱, 恨两种原则.

至于爱恨两种原则如何使四元素构成世界, 他并没给出说明. 但是从他对于生物观点来看, 他的构成论是相当随意的.

他认为生命是这样构成的: 在虚空中漂浮着各种身体零件, 它们随机结合形成了这个世界的生物. 如果人脑袋碰上了人身体, 就形成了普通的人; 如果牛脑袋碰上人身体, 就形成传说中的牛头人.

对此亚里士多德批评道 "存在为了本质, 而非本质为了存在".

# 阿那克萨哥拉

在恩培多克勒那里, 为了保持无素的同一性, 世界变成了元素间的机械组合: 世界只是元素的随机碰撞结合. 而阿那克萨哥拉的世界克服了这种机械性.

阿那克萨哥拉说, 世上的一切都是种子发展出来的, 金就是由金的种子发展而来, 土就是由土的种子发展而来.

至于金的种子是如何发展成现实的金, 土的种子是如何发展现实的土, 阿那克萨克拉说, 是由于世界的理智.

这样看来, 柏拉图的相论 ( 理念论 ) 简直就是颠倒过来的阿那克萨哥拉的种子说. 只是在柏拉图那里, 美的事物是对美的相的分有, 是一个自上而下的下降过程, 在阿那克萨哥拉那里, 金的种子发展成金, 是一个自下而上的上升过程.

然而虽然说种子发展成世界是因为理智, 但在具体解释世界的生成时, 并没有把世界的理智如何起作用, 以至于理智在他的学说中, 最终变成了无用的东西.

# 原子论

如果说阿那克萨哥拉是对恩培多克勒机械世界观的克服, 那原子论就是对恩培多克勒机械世界的进一步发展.

原子论有两个代表人物, 分别是留基伯和德谟克利特.

他们认为世界是由微小, 坚硬, 不可分的原子构成. 原子之间只有形状, 排列和位置之间的区别.

如果从唯物主义的角度来说, 原子论算是最早, 最彻底的唯物主义了. 这原子论的世界里完全没有神存在.

然而关于原子论有一个有一个有趣的公案. 在古希腊哲学中如此独特的原子论, 柏拉图的作品里完全没有提到过, 就好象他完全不知道有这样一种学说一样. 至于柏拉图不提原子论的原因, 至今哲学史家也没有达成共识.

# 智者

智者在古腊哲学中算不得一个学派, 因为其实他们什么都不主张.

他们只是将古腊希哲学智慧技术化罢了, 他们收钱授徒, 教人 "以弱的道理胜过强的道理" 的技术. 他们的兴起的主要原因是当时雅典民主制, 辩论术是一种关键的技能.

据说, 有一个智者在广场上演讲, 主题是世界上最坏的东西是人的舌头, 在场听众听得恨不能当场把舌头割了; 第二天这个智者又在同一个地方演讲, 主题是世界上最好的东西就人的舌头, 同一批听众听得恨不得回家把舌头供起来.

但是任何人做事都要证明自己的合理性. 智者了证明自己的合理性, 提出 "人是万物的尺度": 人觉得什么是对的, 什么就是对的, 因此关键的问题就是如何影响人, 让他们认为什么是对的.

# 希腊古典哲学

希腊古腊哲学才是古希腊哲学的正餐, 我们甚至可以把整个前苏格拉底哲学看作是希腊古典哲学的序章.

希腊古典哲学指的是苏格拉底, 柏拉图, 亚里士多德三圣哲的哲学.

希腊古典哲学博大精深, 决不是一个极简史可以梳理出框架和脉落的. 因此我们在这里仅仅给出一条线索, 试着能简单的把希腊古典哲学串一串.

# 线索

我们从伊奥尼亚哲学顺下来, 可以看到古希腊哲学家总是试图找到一个范畴来对世界进行统一的解释.

伊奥尼亚学派提出 "始基", 到了南意大利派有了 "无素", 恩培多克勒在 "元素" 上加上了 "原则", 阿那克萨哥拉将 "元素" 发展为 "种子", 原子论则又发展出 "原子".

而什么都不主张的智者们, 其实是彻底放弃了通过找到一个范畴来对世界进行统一解释的路线, 而是采用了 "人是万物尺度" 的相对主义路线.

而到了希腊古典哲学, 苏格拉底意识到, 我们要找到一个范畴来对世界进行解释, 那么我们是不是先要找到一个可以合理的定义范畴的方法啊. 所以从苏格拉底开始探讨如何下定义的问题, 而这个问题一直贯穿到亚里士多德.

如何下定义, 就是我们这里给出的线索.

# 分期

苏格拉底没有著作留世, 我们对于苏格拉底哲学的了解, 主要从柏拉图的对话录来.

我们对柏拉图对话录分为早期, 中期, 晚期三部分. 通常认为, 柏拉图早期作品是转述老师苏格拉底的哲学. 中期作品开始主要表达自己的相论 ( 也被翻译为理念论. 但为认为翻译为 "相论" g更好 ). 到了晚期他开始对自己的相论进行反思, 提出了一些新的思想.

亚里士多德的存世作品被认为是他的学园讲课的教材或授课笔记. 其中很多观点不一致之处, 是因为他的思想也分为早期和晚期, 而他早期思想和晚期思想也有不同之处.

因此我们这里的分期是这样的:

  • 苏格拉底哲学
  • 柏拉图相论
  • 柏拉图晚期
  • 亚里士多德早期
  • 亚里士多德晚期

# 苏格拉底哲学

关于苏格拉底为什么开始他的哲学研究的故事这里就不多说了, 我们还是围绕我们线索: 如何下定义.

苏格拉底下定义的方法是: 辩证法. 所谓的辩证法, 大概是这样一个方法: 他会问一个自认为理解某个概念的人, 他所理解的那个概念是什么意思, 比如问一个自以为理解美的人, 什么是美. 当这个人说出自己的定义后, 苏格拉底会挑战这个定义的边界, 通常是找到反例, 以迫使回答的人修改自己的答案, 而苏格拉底又会再次挑战这个概念的边界, 于是答案又会再一次被修改.

这个过程通常会进行好几轮. 最终的结果通常是, 被提问的人承认, 自己原来认为理解的概念, 其实并不知道它是什么意思.

于是苏格拉底会提出, 看来真正的知识只属于神.

# 柏拉图的相论

在柏拉图早期转述苏格拉底哲学的对话录中, 对话录讨论的问题往往得不到真正的答案, 对话录的结尾以对话人承认自己的无知结束.

而在柏拉图中期的对话录中, 柏拉图提出了自己的定义方法. 这个方法就是相论.

所谓相论就是说任何现实东西都有一个完满的相, 现实的事物是对于相的分有. 比如美的事物都是对美的相的分有.

# 柏拉图晚期哲学

柏拉图晚期开始对自己提出的相论进行反思, 首先是 "相" 这相概念本身, 怎么看, 都只有 "真善美" 这样好的东西才有 "相", 那 "假丑恶" 这样的不好东西也有 "相" 么?

其次 "对相的分有" 这相观点也可疑了起来: 现实分有相是怎么分有呢, 是像分蛋糕一样你一块我一块的分? 那完满的相不是越分越不完满了? 还是像分账篷一样东一块西一块分?

在《巴门尼德》篇的前半部分, 柏拉图列举了相论的问题. 在《巴门尼德》篇后半部分, 柏拉图试图找到一种新的理论. 这新的理论叫做通种论.

通种论是认为世界是由几个最通用的 "种" 结合成的. 在《巴门尼德》中柏拉图讨论了几个最通用的 "种" 的结合的情况.

在更后期的作品中, 柏拉图还是出了二分法. 所谓二分法就是通过不断对概念进行进二分来进行定义的方法.

比如, 那个著名的笑话:

  • 事物分为生物和非生物, 人属于生物;
  • 生物分为有足生物和无足物, 人属于有足生物;
  • 有足生物分为两足生物和多足生物, 人属于两足生物;
  • 两足生物分为长毛的和无毛的, 人属于无毛两足生物;
  • 所以人是两足无毛生物.

# 亚里士多德早期

我们看到二分法的主要问题是, 在选择二分的时候, 随意性太强. 因此, 亚里士多德在 《范畴篇》中提出了他的定义方法: 属加种差.

在《范畴篇》中亚里士多德提出了实体的概念. 他说, 实体是 "被表述而不表述其他东西的东西". 比如我们说, 苏格拉底是高个子. "苏格拉底" 是被 "高个子" 表述的, 我们只能用 "高个子" 来表述 "苏格拉底", 而不能用 "苏格拉底" 表述 "高个子".

然而我们发现, 有的实体比别的实体更是实体, 比如, 我们可以说, 人是智慧的, 也能说, 苏格拉底是智慧的. 但是 "苏格拉底" 就是比 "人" 要更是实体的实体, 因为我们还可以说 "苏格拉底是人".

所以亚里士多德说, 那些比别的实例更是实体的实体叫做第一实体. 第一实体就是那些最最具体的东西, 比如 "苏格拉底". 别的可以表述第一实体的实体叫第二体. 第二实体就是那些第一实体的种类, 比如 "人". 至于那些不是实体的, 叫做属性, 比如 "高个子" 或 "智慧".

所以我们可以怎么下定义呢? 就是实体的第二实体加上实体与其他实体的差异. 比如我们怎么给苏格拉底下定义呢? 苏格拉底 ( 实体 ) 是最智慧 ( 与其他实体的区别 ) 的人 ( 第二实体 ).

这就是 "属加种差" 定义法.

# 亚里士多德晚期

亚里士多德后来提出了四因说: 世界上有四种原因, 质料因, 形式因, 动力因和目的因.

这时他早期的实体论发生了变化. 他问, 质料和形式相比, 哪个更是第一实体呢? 如果按照他在范畴篇中的看法, 那个最最具体的才是实体, 那除去所有属性的质料才是实体.

但是对于一个木桌子来说, 倒底是木头使桌子是其所是呢还是桌子的形式使桌子是其所是呢? 那当然是桌子的形式使桌子是桌子了.

于是他说, 质料因与形式因相比, 形式因才是第一实体.

以上就是我所能用最少的字给出的极简古希腊哲学史了.

最后来聊聊, 如果想了解古希腊哲学哪些书可以读.

首先是苗力田的《古希腊哲学史》, 这是一本古希腊哲学的史料汇编. 读这本书可以建立关于古希腊哲学基本事实的知识框架.

如果有余力可以看看汪子嵩主编的《古希腊哲学史》, 厚厚的四卷本, 非常详细.

如果没有时间和精力看这套《古希腊哲学史》, 可以看叶秀山的《前苏格拉底哲学》和《苏格拉底哲学》. 不厚, 还格外的好读.

柏拉图与亚里士多德都有中文全集出版. 柏拉图全集是王晓朝译的, 他还写过《教父学研究》, 设过奥古斯丁的《上帝之城》. 亚里士多德全集苗力田组织编译的, 但据说是组织大学生译的, 有的地方读起来是有点奇怪.

如果想快速了解亚里士多德的思想, 可以读汪子嵩的《亚里士多德关于本体的学说》.

这就是我学习古希腊哲学的一点心得, 希望可以帮到斯奥同学和其他想了解古希腊哲学的朋友.
