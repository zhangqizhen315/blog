# 要牙签的乞丐

有这么个笑话:

饭店中一桌食客正在吃饭, 突然有个乞丐突然跑进来. "老爷们行行好, 施舍根牙签吧." 食客们都是体面人, 怎么会舍不得根牙签, 自然是大方施舍了. 但没想到接二连三的有乞丐来要牙签. 食客们都一头雾水, 不知缘故. 直到又有一个乞丐跑进来. "老爷们行行好, 施舍根吸管吧." 食客们好奇 "咦, 别人都要牙签, 怎么只有你要吸管?" 乞丐着急的说: "刚才有人喝多, 在门口吐了. 那些要到牙签的已经把干的挑完了, 老爷们赏根引管, 我还能落点稀的吃."

一段时间以来, 总觉得身边的人 -- 包括现实交往到的人和网上的网友 -- 有种对知识或道理挑剔的态度: 无论是什么道理, 总想找出它何以不起作用的说法来. 形而上到维特根斯坦, 形而下到 pmp 项目管理理论, 凡能说出 "道理" 的, 都能被说成是陈词滥调. 每听到这种高论, 我总要强忍住去讲 "如人饮水, 冷暧自知" 的道理: 人要从道理中受益, 总是先努力试着把道理领受成有用的才行, 就如人喝水, 是凉是热, 能不能咽到吐子里, 总是自己决定的. 然而终于, 我还是忍往没讲这个道理. 因为这毕竟是个道理, 是道理, 就总能被说成是陈词滥词: "如人饮水, 冷暧自知" 嘛, 道理起不起作用, 总要看听的人听的进听不起了.

直到有一天我想起这个笑话, 就释然了. 我这是以乞丐之心度食客之腹了. 对于馈乏如乞丐的人, 无论什么东西, 总要试着从中找出价值来, 哪怕是吐出的秽物, 也要挑干的吸稀的利用起来. 所以对于道理, 我总试着去起它何起会起作用， 假设在哪种情况下我能用上. 至于那些丰沛到要往出吐的食客, 他们多见到一点, 都觉得恶心.

有的人的盛宴, 别人看来只是战场.

# 洗地的福音

在知乎上看到一个说法: 《圣经》的福音书是在给罗马人洗地.

依据是《玛窦福音》最后三章里, 写耶稣受难时, 每每写到罗马总督彼拉多对处死耶稣的态度, 总是不情不愿: 先是写彼拉多的老婆梦到耶稣是义人, 不让他判耶稣死刑; 再讲彼拉多 "原来他知道，他们是由于嫉忌才把他解送来的"; 等到判了耶稣死刑, "就拿水，当着民众洗手说：'对这义人的血，我是无罪的，你们自己负责吧！'". 似乎彼拉多是被犹太人逼着杀了耶稣.

但是, 犹太人处死人的方式是石刑 -- 即用乱石砸死 -- 而钉上十字架, 是罗马人处刑的方式. 这事怎么看, "从性质上讲", 都像是具有统治地位的罗马人处死了具有革命性的新兴宗教领袖. 但《圣经新约》却非常诡异将此事塑造成罗马人在犹太领袖的压力下, 处死了耶稣. 似乎是在有意的将犹太人与统治他们的罗马人的矛盾转移成犹太人自己之间的矛盾. 再考虑到后来罗马将基督教立为国教的史实, 这一段关于罗马人杀死耶稣的描述就很像是在为罗马人洗地了.

不得不说, 这种说法很具有娱乐性. 让我想起一本非常喜欢的书 -- 《剑桥倚天屠龙史》 -- 这书假设《倚天屠龙记》是史实, 以历史研究的形式, 以明教的发展隐喻了中国近代史, 里边的道理能自圆其说, 娱乐性也很足, 只是我们知道《倚天屠龙记》不是史实.

其实, 只要我们能够以 "历史主义的态度" 看《圣经》 ( "历史主义" 是我学咕哒的舌, 其实我并不知道 "历史主义" 是什么意思, 我估且把这个词理解成 "以看历史的方法和态度来看《圣经》" ), 我们完全不用这种过于戏剧化和阴谋论的眼光来看待这个事情. 罗马人当时, 对于犹太人的宗教, 完全是一种蒙逼的状态.

我们知道, 犹太人要求罗马人处死耶稣的理由是, 耶稣自称是天主子. 这对于罗马人来说, 自称神之子怎么就要被处死了? 在希腊罗马的传统中, 是个贵族就说自己是神的后裔, 这个家族来源于宙斯, 那个家族来源于波塞冬, 实在正常不过了. 甚至在他们的观念里, 一个人如果不是神的后裔, 他凭什么成为贵族呢?

从这个角度, 我们完全可以理解, 为什么在彼拉多看来耶稣可以是一个义人. 甚至可以理解, 当他让犹太人从杀人越货的大盗巴辣巴和耶稣中选一个释放另一个处死, 而犹太人选择释放巴辣巴处死耶稣时, 他震惊心情, 以至于要当众洗手, 表示无罪.

当然《圣经新约》是在给罗马人洗地这个说法, 格外符合 "普世宗教是在为帝国主义洗地" 的叙事, 因此在某些人那里格外具有说服力. 就如因为可以隐喻近代史, 《倚天屠龙记》者可以被当史实一样.
