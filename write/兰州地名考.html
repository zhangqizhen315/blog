# 兰州地名考

# "城" 与 "关"

"甘肃" 是甘州与肃州的合称. "肃州" 是酒泉. 汉武帝时霍去病给匈奴 "去病" 的时候 ( 西北话里, "去病" 一词有收拾坏毛病的意思, 比如大宝如果老是计算出错, 我就可以对她说: "你是不是想让我给你去去病." ), 在此地大胜. 于是武帝赐酒以赏. 霍去病以功在全军, 人多酒少, 遂倾酒于泉中, 与将士共饮, 自此便有了 "酒泉" 之名. 霍去病接着往西打, 在一处叫 "流沙" 的地方大破匈奴, 武帝在此置郡, 取名 "张掖", 是 "张国之掖, 以通西域" 的意思. 大概是说, 在此之前, 汉朝只能夹着胳膊做人, 畏手畏脚; 此后终于能把手臂伸展, 张开掖窝了.

甘肃这个地方, 自古就与打仗少不了干系. 兰州做为甘肃的省会, 虽在甘州, 肃州以东, 但也是军事重镇. 兰州早先叫 "金城郡". 为何得名 "金城", 我听过两种说法, 一种是说 "筑城得金, 故曰金城", 一说是 "称金, 取其固坚固也". 总之, 最初是一个军事要塞, 固若金汤. 据说在王莽时期, 青海湖附近的西海郡被羌人攻占, 王莽为了与东海郡, 北海郡, 南海郡凑足 "四海之数", 金城郡也被改成过西海郡. 后来随着东汉和羌人的作战, 金城郡被争来夺去. 直到魏晋时期, 才大概算是消停. 直到隋朝金城郡才改为 "兰州总管府".

"先有金城关, 后有兰州城." 金城关据说就是金城郡时期留下的地名. 当年金城关在黄河以北依山而建, 山河之间只有一条大路. 只要固守此处, 真是一夫当关万夫莫开. 当然, 现在的金城关早已没了关隘城墙, 只有现代修的仿古建筑, 作为文件风情区. 现在金城关一带有很多回族兄弟居住, 沿河有一座水上清真寺, 底部造形如船, 上有阿拉伯拱顶清真寺.

金城关往东不到一里, 就是号称 "黄河第一桥" 的 "中山桥". 中山桥的前身是 "镇远浮桥", 建于明朝洪武年间. 光绪三十一年, 甘肃洋务总局与德商签定合同, 在此建了铁桥. 民国时, 为了纪念孙中山, 改名为 "中山桥". 中山桥通体铅灰色, 是抗日战争时为了躲避日军轰炸涂的. 在此处建桥, 是因为此处河谷宽阔 "河水少缓". 我小的时候常常趴在中山桥的栏杆上低头看河水, 盯得久了, 就有桥在动的错觉.

过了桥是老兰州城的桥门. 进了桥门就进了老兰州城. 但现早没了城墙, 也没了什么桥门. 甚至连桥门这个地名都都没留下. 取而代之的是中山路, 应该是由中山桥而得名.

沿中山路往南是西关什字. 西关十字应该是老兰州城的西门, 出了西关十字再往西是七里河区, 往东是城关区. 我还住在兰州时西关十字是个公交枢纽, 现在可能改成地铁站了吧. 2000 年前, 西关十字公交枢纽白天停公交车, 晚上公交车都加厂了, 就会摆出夜市来, 全是各种小吃. 那时我还在上初中, 晚上偶尔去吃. 记得有一家米线非常好吃, 特别是里边的酸菜格外的香. 后来夜市被撤销了, 好像是因为出了治安事件.

西关十字公交枢钮边上就是西关清真大寺, 兰州最大的清真寺. 西关清真大寺再往西, 叫做萃英门, 以前是举院, 考举人的地方. 以前甘肃人要考举人, 要到陕西考. 左宗棠任陕甘总督时, 看到甘肃人去陕西考试费时费力, 就向朝庭奏请 "陕甘分闱". 准奏后就在此建了举院. 据说老兰大也在此处. 萃英门就是取 "荟萃精英" 的意思.

萃英门旁边是西稍门, 解放兰州的时候, 解放军从这里打进的兰州, 所以现在改名叫解放门了. 我在兰州时, 家就住在解放门. 可惜住在举院和兰大旧址处并没有使我学习变好. 但也不能怪我, 解放门早已是一个大杂院, 有房管所, 交通局等好几个单位的人住在里边, 一点也没有 "荟萃精英" 的感觉.

过西关十字沿中山路往南走一站公交车的路, 就到了双城门. 当年抗日战争时期, 日军经常轰炸兰州, 为了疏散群众, 在此处城墙开了两个简易城门, 因此得名双城门. 至此中山路就到了头, 再往西南去是白银路, 往东去叫庆阳路. 顺白银路而上就到安定门. 安定门旧称永康门, 寓意安宁康定, 是老兰州城西南角的城门. 出安定门再向南, 是就了皋兰山. 顺着庆阳路再往东过两个红绿灯就到了南关十字. 与西关十字类似, 南关十字是老兰州城的南门.

自双城门到南关十字一片是古时的兰泉驿. 兰泉驿内建有住房和马厩, 专供过往投递公文的驿卒住宿, 换马. 而今此处早是永昌路百货和金城剧院.

西关十字往东是张掖路商业步行街, 沿街尽是商铺餐厅. 步行街到尽头是中央广场. 有传说兰州是中国的几何中心, 中央广场是兰州的几何中心, 不知依据何在. 但在某种意义上, 中央广场确实是兰州的政治中心, 因为此处是省政府的所在地. 省政府的前身是明肃王府, 建于明惠帝建文元年, 自那时起, 此地就是就是甘肃省最高行政权力机构所在地. 清朝时, 此地改为陕陕甘总督府, 民国时改为甘肃省政府.

省政府前有小花坛, 花坛中心是国旗杆. 据说这旗杆就是那几何中心的几何中心 -- 都市传说总是这样有鼻子有眼的. 我初高中上学要经过此地, 早上要赶上升旗, 就能看到路上汽车停下观礼.

除了省政府, 这里还有三个书店. 两个国营的: 新华书店和古藉书店. 一个私营的: 广场书城. 我的第一份工作就是在广场书店当店员.

省政府向南去的道路叫酒泉路, 顺酒泉路向南就到了前边说的南关十字. 新华书店边向北的道路叫通渭路, 通渭路向北就到了黄河. 这里有一座新修的大桥叫通元大桥.

省政府再往东张掖路就到了头, 再向东的路叫秦安路, 南北向与之相关的叫静宁路. 张掖路与静宁路相交之处就是老兰州城北门之所在.

我初高中上学的兰州三中就在秦安路. 秦安路再向东就到了广武门. 广武门古时又称新关门. 所谓新关门, 是指此关门是在建城以后所建, 相较城门为新关门. 兰州的练兵场, 教场设在此处. 因此到了清朝重修此门时就正式定名为广武门了.

仅从地名来看, 老兰州城就也就这么大了: 东起广武门, 西至西关十字, 北临黄河, 南到南关十字. 说来有趣的是, 我初高中时, 家住西关以外的解放门, 上学在广武门的三中. 每天上下学正好横穿老兰州城. 我每天步行上下学, 路上大概也就需要三十分钟左右.

如今兰州早已不见了城墙与关门, 只能依稀靠着地名来想象当年兰州城千百年间的生活: 络绎不绝的使团商旅自西域或中原而来, 经金城关过镇远桥, 进桥门而入; 或由安定门广武门进兰州城; 先去兰泉驿放下行李, 安顿货物, 拴好骡马, 再去肃王府办好公事, 如还有余裕便呼朋引伴到繁华的大街游玩; 直到夜幕降临南来北往的旅人们回到兰泉驿, 相聚而饮, 把酒言欢.

兰泉驿旧址所在的永昌路, 现在是一条夜市. 每到天黑自黄河边直到皋兰山下, 一路上全是卖衣帽鞋袜和各种小玩意的小摊, 逛夜市的人熙熙攘攘摩肩擦踵. 南关十字曾经有个啤酒广场, 人们聚在此处喝酒阵势甚大, 被人戏称为 "万人坑". 当年在万人坑中喝酒的人们, 在觥筹交错间, 用醉眼望向以前的兰泉驿现在的永昌路夜市, 那人头攒动的盛景中, 恍忽不知几人在今几人为古, 古今交错似在一瞬瞥见了永恒.
