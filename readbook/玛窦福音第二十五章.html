# 玛窦福音第二十五章

# 十童女的比喻

# 经文

1 “那时，天国好比十个童女，拿着自己的灯，出去迎接新郎。2 她们中五个是糊涂的，五个是明智的。3糊涂的拿了灯，却没有随身带油；4 而明智的拿了灯，并且在壸里带了油。5 因为新郎迟延，她们都打盹睡着了。6 半夜有人喊说：新郎来了，你们出来迎接吧！7 那些童女遂都起来，装备她们的灯。8 糊涂的对明智的说：把你们的油，分些给我们吧！因为我们的灯快要灭了！9 明智的答说：怕为我们和你们都不够，更好你们到卖油的那里去，为自己买吧！10 她们去买的时候，新郎到了；那准备好了的，就同他进去，共赴婚宴；门遂关上了。11 末后，其余的童女也来了，说：主啊！主啊！给我们开门吧！12 他却答说：我实在告诉你们：我不认识你们。13 所以，你们该醒寤，因为你们不知道那日子，也不知道那时辰。”

# 《新约导读系列 -- 玛窦福音》

这个比喻没有新娘, 因为童女象征着教会, 就是基督的新娘, 而基督是她默观中的爱情对象. 旧约里, 黑夜是审判的象征, 因为新郎不在. 新郎的光临, 就是白昼降来的预报.

# 《新学院村圣经诠释 -- 玛窦福音》

第二个比喻 ( 此本导读, 是将上一章最后一个段落的比喻与本章的诸比喻作为一个系列来看, 而事实上我们也确实可以看到上一章主题与本章主题的连续性 ), 是劝诫人们为人子的来做好准备. 比喻中, 耶稣被塑告成新郎的角色 ( 犹如: 玛九 15; 并参阅: 依五四 5; 耶三十一 32; 欧二 16, 其中上主雅威是以色列的新郎 ). 在耶稣时代, 婚礼分为两个阶段进行, 首先是在新娘的父家举行订婚仪式, 新娘向末来岳父呈上结婚的婚约以及聘礼; 新娘会继续居住在父家直到第二階段, 即新娘一年后嫁去夫家. 这第二阶段, 就是这个比喻描述的階段, 当新郎和新娘的父亲在做最后交涉时, 童女们在等待新郎的到来，一旦最后协定达成共识, 婚礼的队伍就出发去新郎家, 婚宴就要开始了.

这里, 正在等待的 "童女" ( 译自希腊字 Parthenos ) 都是新郎的朋友; 新娘在故事中从未被提及. Parthenos 这个希腊字, 是指一位适婚年龄 ( 十二岁左右 ) 的 "童贞女子". 明智和糊涂童女的对比, 令人想起那位 "建造房屋的人" ( 玛七 24 ~ 27 ). 童女们带的究竟是 "浸泡着油布所包裹着的火炬" ( 这通常是指当时的 "灯具" ), 还是 "带有把柄的灯芯油灯", 这里并没有表达得很清楚.

玛五 14 ~ 16 提供了一个线索, 可以说明为什么童女不能分享她们的油: 灯光等同于他人看得见的好行为, 并引领人赞美称颂天主. 同样, 玛七 24 ~ 27 也指出: 聪明人是那些听了耶稣的话而实行的人. 正如比喻中明智的童女, 她们就是那些忠实地为末世做好准备的人. 没有人可替他人做准备. 末世到来时 ( eschatological moment ), 人只有 "准备好了", 还是 "没有准备好" 两种区别而己.

# 《偕主读经 -- 玛窦福音》

四位圣史中, 唯有玛窦记载了这个比喻, 指的是耶稣的再度来临. 所描述的是人们期待复活, 并等候基督再次来临的情况. 天国不是被比喻为十个伴娘, 而是被比喻为那个盛大的婚宴. 这个比喻的中心讯息是让人充分准备好自己.

比喻提到两个事实, 即新郎迟迟未到, 和等候者睡着了 (不警醒). 那些愚蠢伴娘的愚蠢之处, 不在于睡着了 (因为她们都打吨儿睡着了). 而在于她们没有准备好随时执行使命. 她们没有考虑到新郎可能会迟到, 因此也没有准备足够的灯油. 那些聪明的伴娘拒绝与她们分享灯油, 这是一种比喻手法, 告诉我们, 这种必须的准备是个人性的, 不可以转借. 同样, 只依赖他人的忠信而自己不努力, 是毫无用途的.

新郎的反应说明这些愚蠢的伴娘象征信徒团体中的徒有虚名、没有全心全力投入的人, 最终不会准备好自己. "所以, 你们要警醒" ( 13 ). 这句话是在警告我们, 要时常保持忠信, 并全心全力将耶稣的教导付诸实行.

# 塔冷通的比喻

# 经文

14 “天国又如一个要远行的人，将自己的仆人叫来，把财产托付给他们：15 按照他们的才能，一个给了五个‘塔冷通’，一个给了两个，一个给了一个；然后动身走了。16 那领了五个‘塔冷通’的，立刻去用来营业，另外赚了五个。17 同样，那领了两个的，也赚了另外两个。18 但是，那领了一个的，却去掘开地，把主人的银子藏了。19 过了多时，仆人的主人回来了，便与他们算账。20 那领了五个‘塔冷通’的上前来，呈上另外五个‘塔冷通’说：主啊！你曾交给我五个‘塔冷通’，看，我赚了另外五个‘塔冷通’。21 主人对他说：好！善良忠信的仆人，你既在少许事上忠信，我必委派你管理许多大事：进入你主人的福乐吧！22 那领了两个‘塔冷通’的也前来说：主啊！你曾交给我两个‘塔冷通’，看，我赚了另外两个‘塔冷通’。23 主人对他说：好！善良忠信的仆人！你既在少许事上忠信，我必委派你管理许多大事：进入你主人的福乐吧！24 随后，那领了一个‘塔冷通’的也前来说：主啊！我原知道你是个刻薄的人，在你没有下种的地方收割，在你没有散布的地方聚敛。25 因为我害怕，所以我去把你的‘塔冷通’藏在地下；看，你的仍还给你。26 主人回答说：可恶懒惰的仆人！你既知道：我在没有下种的地方收割，在没有散布的地方聚敛；27 那么，你就该把我的银子，交给钱庄里的人，待我回来时，把我连本带利取回。28 所以，你们把这个‘塔冷通’从他手中夺过来，给那有了十个‘塔冷通’的，29 因为凡是有的，还要给他，叫他富裕；那没有的，连他所有的，也要由他手中夺去。30 至于这无用的仆人，你们把他丢在外面的黑暗中，在那里必有哀号和切齿。”

# 《新约导读系列 -- 玛窦福音》

现在, 基督信友们要得到以下的教训. 注意: 财产是按每个人的才能赋给的 ( "塔冷通" 原是钱币的单位, 因为这个比喻, 它 ( Talent ) 开始指 "天才" ) 而他们的赏报不是财富, 而是更大的职责. 大量的金钱被称为 "少许事", 与天上的 "许多大事" 相对, 因为这两件事可能象征这世界的一切财物是个信托基金, 而富人必须为基督来管理, 分配它们.

# 《新学院村圣经诠释 -- 玛窦福音》

这个比喻通常被认为是劝免人充分使用天主赐给的因宠. 然而, talanton ( 塔冷通 ) 这个希腊字, 除了是货币单位或重量测定外, 并无其它涵义. 这个比喻中, "塔冷通" 是表示一大笔线. 这个比喻所描述的是: 两个仆人把主人委托给他们的金钱, 用于投资并获得双倍的利润, 使他们赢得了主人对他们的认可, 也分享了主人的喜悦, 并被主人赋予更大的责任; 然而, 第三个仆人把金钱埋在地下, 这在古代被认为是保存贵重物品的最佳方式, 但他却得到主人苛刻的处罚.

耶稣并非生活在一个认为 "投资可增加财富" 的资本主义社会中, 这是理解这个比喻的关键. 当时人们怀着的, 是有限资源的想法: 财富是有限的, 增加一个财富, 便是另一个人的损失. 农夫的生活目标, 是足以养活家庭. 任何人为自己集聚大量财富, 会被视为贪婪和邪恶. 所以, 假如这个比喻里的主人指的不是天主的话, 那么, 第三个仆人就是最值得尊敬的的一位了, 因为只有他拒绝与 "贪婪无止境的主人" 合作. 这里是在警告有钱人停止剥削穷人, 并鼓励穷人要勇敢地采取行动揭露贪婪的罪孽.

最后一节相当严肃, 是在描述 "反抗富人和强权的人", 将会发生的悲惨后里: "把他丢在外面的黑暗中, 在那里必有哀号和切齿". 这也在鼓励门徒: 当他们面对不公平的制度时, 要团结一致, 共同找寻抵制的方法. 机会总是有的, 因为末日尚未到来.

# 《偕主读经 -- 玛窦福音》

这个比喻也劝导人, 在天主审判的时刻到来之前, 应该保持忠信和警醒.

从上下文中我们可以看出, 这个比喻的重点是仆人们交账的情形, 尤其聚焦于那个过分谨慎的仆人身上. 耶稣斥责那些摇摆不定的人, 他们接受了天国的讯息, 后来却躲在空洞的安全感中, 没有结出果实.

耶稣的门徒必须要使天国的资产在指定的时期内结出果实. 为玛窦而言, 这是指 "教会的时期". 那些没有结出果实的人, 即使他们能将这些资产保存得完好无损, 但在最后审判时仍将全部失去. 同样, 教会的团体也必须要警惕、小心, 以免将自己的使命沦为一种敷衍塞责的自满自足.

# 公审判

# 经文

31 “当人子在自己的光荣中，与众天使一同降来时，那时，他要坐在光荣的宝座上，32 一切的民族，都要聚在他面前；他要把他们彼此分开，如同牧人分开绵羊和山羊一样：33 把绵羊放在自己的右边，山羊在左边。34 那时，君王要对那些在他右边的说：我父所祝福的，你们来吧！承受自创世以来，给你们预备了的国度吧！35 因为我饿了，你们给了我吃的；我渴了，你们给了我喝的；我作客，你们收留了我：36 我赤身露体，你们给了我穿的；我患病，你们看顾了我；我在监里；你们来探望了我。37 那时，义人回答他说：主啊！我们什么时候见了你饥饿而供养了你，或口渴而给了你喝的？38 我们什么时候见了你作客，而收留了你，或赤身露体而给了你穿的？39 我们什么时候见你患病，或在监里而来探望过你？40 君王便回答他们说：我实在告诉你们：凡你们对我这些最小兄弟中的一个所做的，就是对我做的。41 然后他又对那些在左边的说：可咒骂的，离开我，到那给魔鬼和他的使者预备的永火里去吧！42 因为我饿了，你们没有给我吃的；我渴了，你们没有给我喝的；43 我作客，你们没有收留我；我赤身露体，你们没有给我穿的；我患病或在监里，你们没有来探望我。44 那时，他们也要回答说：主啊！我几时见了你饥饿，或口渴，或作客，或赤身露体，或有病，或坐监，而我们没有给你效劳？45 那时，君王回答他们说：我实在告诉你们：凡你们没有给这些最小中的一个做的，便是没有给我做。46 这些人要进入永罚，而那些义人却要进入永生。”

# 《新约导读系列 -- 玛窦福音》

玛窦利用旧约的意像来描绘最后审判, 并在其内强调基督的天主性. 耶稣现在成为 "君王", 这个称号以前只加天主父的 ( 五 34 ~ 35 ). 这次审的对象是一切恶人与义人, 必须以全人类的复活为先决条件. 耶稣, 上主的仆人, 说他自己就是穷人和受苦的人 ( 由此可见, 保禄所说 "基督奥体" 的道理, 是从耶稣自己的教训发展而来的 ). 兄弟间的友爱是最后审判的准绳.

# 《新学院村圣经诠释 -- 玛窦福音》

这是玛窦圣史记述的最后一个比喻, 也是《玛窦福音》独有的. 当人子光荣地来临时, 末世审判早已来了 ( 31 节 ). 这个情景, 与耶稣命令追随者去 "使万民成为祂的门徒" ( 玛二十八 16 ~ 20 ) 有密切的联系, 这个比喻假定比命令已被履行. 那里的 "万民" ( 二十八 19 ) 是希腊文 panta ta ethne 的中译词, 意思就是本比喻中的 "一切民族" ( 32 节 ). 现在 "一切民族都被聚集起来", 向天主交账.

为什么要把 "绵羊" 与 "山羊" 分开来? 原来不清楚. 其实, "绵羊" 与 "山羊" 都极为珍贵. 没有证据显示: 白天时, 让绵羊与山羊一起吃草, 到晚上, 牧羊人却要把他们分开. 《玛窦福音》中, 在末日把被选的与遭弃的分开的图像很多 ( 请见: 玛三 12; 十三 24 ~ 30, 47 ~ 50; 二十四 40 ~ 41; 二十五 1 ~ 13 ). 因为大多数人是右撇子, 右手比较有力, 灵活, 右侧也就成为钟爱, 受祝福及荣誉的象征.

耶稣的图像在这里, 从牧羊人转换成君王 ( 34 节; 参阅: 玛二 2; 二十一 5 ). 同时就像梅瑟将 "祝福" 和 "诅咒" 摆在以色列子民面前, 给他们选择 ( 申十一 26 ) 一样, 耶稣也将那些 "我父所祝福" ( 34 节 ) 与那些 "可咒骂的" ( 41 节 ) 分开, 这是预先定好的, 而是天主向所有发出邀情 ( 玛五 45; 十三 3 ~ 9 ), 接受与否全冯个人选择. 接受了邀请的人, 这些人的善行是可见的, 祝福和天国的产业在期待着他们.

耶稣说过 "天国的福音必先在全世界宣讲, 给万民作证, 然后结局才会来到" ( 玛二十四 14 ); 由这句话得知: 玛窦圣史很可能正期待着伟大福传使命 ( 玛二十八 16 ~ 20 ) 的完成; 到那时, 包括以色列人, 外邦人和基督徒在内的万民, 都已听到福音, 并根据他们的行为在接受审判.

"最小的兄弟" ( 40 节 ) 和 "最小中的一个" ( 45 节 ), 在玛窦的眼中很可能是指有此需要的基督徒弟兄, 而不只是指任何一位有需要的人. 《玛窦福音》中的 "小子" ( 玛十八 6 ~ 14 ) 和 "最小的" ( 玛十一 11 ), 是指基督徒团体里需要帮助的弱势成员; 况且, 耶稣曾许诺要给予那些接待需要帮助的 "先知", "义人", 或 "门徒中的小子" ( 玛十 41 ~ 42 ) 的人赏报. 人怎样接纳耶稣, 是透过怎样接纳传佈福音的宣讲来做审判的依据.

# 《偕主读经 -- 玛窦福音》

这篇道理的目的不是为描述末日的事件, 而是在谆谆劝导人准备好自己, 只有这样, 才能够经得起最后的考验. 同时也突出其核心讯息, 即耶稣, 这位人子的君王身份. 那些被接到天国里的人, 就是那些以富有同情心的爱对待近人的人.

在《旧约》中曾经提到这六种爱人的方式 ( 依 58:7 ; 约 22:6-7 ). 在这里, 这些方式是爱的诫命的具体实践 ( 22: 37 -40 ) . 耶稣的教导特别反对买卖型的心态: 做某些事的动机纯粹是想获得天主的赏报, 自己可以使天主没有其他选择, 只能赏报他们. 因此, 即使完成这些工作, 也不是为了天主, 而是反对天主, 就等于绑着天主的手, 强迫祂去赏报这些信徒. 这其实是将真正的宗教信仰完全扭曲了.

所以说, 一个人无论以何种善行服务有需要的近人, 其最后的判决还要取决于这些善行背后的动机. 出自真爱的慈善工作, 不会受到任何限制, 也不会为这工作的价值附加任何条件.

耶稣所讲的话针对所有的人, 门徒们所组成的有形可见的团体没有任何区别. 如此说明, 在祂的教会之外, 天主的国度仍然可能实现. 地上的教会不能等同于天国. 教会是服侍天国的谦卑仆人, 而天国可以超越教会可见的范围. 这使我们明白, 原来, 许多人没有听说过耶稣这个名字, 但他们实践了天国的价值, 爱护邻人. 这也就是我们通常所说的 "医名的基督徒". 这一切便能决定他们的最终归宿 (40、 46). 这位审判普世的主, 隐藏于所有的穷人当中, 隐藏在所有痛苦的面容背后:而这种隐藏的事实, 要在最后的时刻显露于众人面前.

另外, 耶稣的这些教训, 也针对那些没有实际投身于自己使命中的基督徒, 为的是将他们从昏睡中唤醒, 警告他们, 在祂第二次到来之前, 每一个人的命运取决于自己对待有急需者的态度.
