# 玛窦福音第十九章

# 婚姻与贞洁

# 经文

1 耶稣讲完这些话以后, 就离开加里肋亚, 来到约但对岸的犹太境内. 2 有许多群众跟随他, 他就在那里医好了他们.

3 有些法利塞人来到他跟前, 试探他说: "许不许人为了任何事缘故休自己的妻子?" 4 他回答说: "你们没有念过: 那创造者自起初就造了他们一男一女; 5 且说: '为此, 人要离天父亲和母亲, 依附自己的妻子, 两人成为一体' 的话. 6 这样, 他们不是两个, 而是一体了. 为此, 凡天主所结合的, 人不可拆散. " 7 他们对他说: "那么, 为什么梅瑟还吩咐人下休书休妻呢?" 8 耶稣对他们说: "梅瑟为了你们的心硬, 才准许你们休妻; 但起初并不是这样. 9 如今我对你们说: 无论谁休妻, 除非因为姘居, 而另娶一个, 他就是犯奸淫; 凡娶被休的, 也是犯奸淫." 10 门徒对他说: "人同妻子的关系, 如果是这样, 倒不如不娶的好." 11 耶稣对他们说: "这话不是人人所能领悟的, 只有那些得了恩赐的人, 才能领悟. 12 因为有些阄人, 从母胎生来就是这样; 有些阄人, 是被人阄的; 有些阄人, 却是为了天国, 而自阄的. 能领悟的, 就领悟吧!"

# 《新约导读系列 -- 玛窦福音》

这次前往圣地结束了对群众的教训, 更为下面全部的叙述预置重要明显的主题.

玛窦简单的描述了耶稣在约旦河对岸的犹太人境内治好病人, 以及 "许多群众" 都跟随着他.

耶稣斥离婚是不合法的. 在基督信仰里, 婚姻要追溯到 "最起初", 因为造人的天主命令人和妻子必该 "合为一体" ( "肉" 在希伯来文原意是 "人" ). 梅瑟法律对天主原先的计划有些修改, 是为了在两件罪恶之中选择较轻微的 ( 因为做丈夫的可能因不准离婚而谋杀妻子. )

宗徒们的一句评论, 就颇消极地把话题引到贞洁问题上来了. 耶稣急欲纠正以方便为应否结婚的准则, 而肯定贞洁是天主贵重的因赐. 这个基督信仰所特有的理想与旧约时的宗教思想背道而驰, 例如依弗大的女儿的故事所显示的. 吾主在此处的口气是典型的闪族语法, 但我们千万不该忽略他给我们绝对的自由来接受这个挑战性的因宠, 以及他所要求的在意念上的纯洁. 在旧约时代, 司祭只从以色列的一支中选出来 ( 肋未 ); 在基督徒中, 这个思宠是赐给那些慷慨大方的男男女女. 同时还要注意一点, 新约圣经, 守贞者成为理想基督徒的象征, 保禄曾表示他意识到在守贞上所拥有的自由, 以及它的无比珍贵. 修会里的贞治誓愿, 例如神父修女的独身生活愿, 是一个积极的行动, 是爱的表示, 是自我奉献给基督. 玛窦福音另一个比喻用贞女来代表教会默观中的爱情, 更加强说明了这一点.

# 《新学院村圣经诠释 -- 玛窦福音》

在耶稣时代, 来往加里肋亚和犹太地区最简捷的路径, 是经过撒玛黎雅. 但玛窦圣史记述耶稣前往耶路撒冷的旅途, 与大多数的犹太人一样, 是绕过约但河东边而直达犹太境界, 为的是要避免经过撒玛黎雅 ( 1 节 ). 这时, 反对耶稣的犹太宗教领袖又以离婚问题向耶稣挑衅, 为的是要试探祂 ( 参阅: 玛十六 1 ). 耶稣不准离婚的言论在 "山中圣训" 中已有教导 ( 玛五 31 - 32 ). 现在提出这个问题, 是要试问耶稣对离婚议题有没有例外法规 ( 3 节 ). 当时 ( 主历第一世纪 ) 的犹太经师传统中有不同的主张, 例如: 霞玛依学派和希肋耳学派之间就常辩论. 希肋耳学派坚信, 男人甚至可以因为妻子煮的菜不好和她离婚 ( 这一派较宽松, 认为任何令丈夫烦扰的事, 都可是构成休妻的理由 ); 而霞玛依学派则争辩说, 只发生淫乱的事 ( 妻子有外遇 ), 才可休妻, 才是离婚的理由.

耶稣回答时, 首先引用《创世纪》中 "那创造者自起初就造了他们一男一女" ( 创一 27 ) 这句话, 然后又引用了《创世纪》的另一句话: "为此, 人要离开父亲和母亲, 依附自己的妻子, 两人成为一." ( 创二 24 ), 从此说明天主创造的旨意, 是要男人和女人成为一体. 耶稣的敌对者, 也引用《申命纪》中的准许人休妻的规定 ( 申二十四 1 ~ 4 ) 来反驳耶稣: 梅瑟充许男人只需要一张休书, 就可与他的妻子离婚. 于是, 耶稣把《创世纪》中天主所启示的旨意及天主不容置疑的命令, 与梅瑟因以色列人无法达到理想标准而做出的让步, 两者之间做出了区隔.

耶稣把休妻形容为奸淫, 除非休妻是基于 "姘居" 的原因 ( 9 节, 也请参阅: 玛五 32 ). "姘居" 这个定的意义不是很明确, 也许指的是性行为不检点, 可能是指 "被犹太法律禁止的通奸行为", 也可能是指 "近亲的婚姻", 那么, 这个问题涉及到了一些希望成为基督徒, 而又处于这种被禁止的婚姻状况中的外邦人. 他们是否应先离婚, 然后才可以加入教会团体呢?

耶稣门徒的反应, 显示耶稣的教导有点极端. 门徒说 "人同妻子的关系, 如果是这样, 倒不如不娶的好" ( 10 节 ), 这句话基实与耶稣前面说过的 "倘若你的手, 或你的脚使你跌倒, 砍下它来 ......, 倘若你的眼使你跌倒, 剜出它来 ......." ( 玛十八 8 ~ 9 ) 一样, 是一种夸张的写作手法; 说 "宁愿把手, 或脚, 或眼睛切除, 而不使其令人跌倒 ( 陷于罪恶 )", 这并不是说耶稣要使所有人的都接受这样的教导. 长久以来, 对于 "有此阄人, 却是为了天国, 而自阄的" ( 12 节 ) 这句话, 到底是指选择保持独身的人, 还是指丧偶或离婚后不再婚嫁的人, 一直争辩不休. 犹太的传统婚姻是符合法律标准和规范的, 仅管有些派别, 如谷木兰隐士等, 他们显然是奉行独身生活的.

基督徒做出这样的独身选择, 其原因是为了宣讲事工, 胜于再度结婚 ( 参阅: 宗九 39, 也许就是这样的情形: 弟前五 3 ~ 16 则论及寡妇行事为人的规范 ). 有关耶稣对离婚的严格教导， 也许是他对那时代的妇女所产生的憐悯和同情的结果, 也是要保障妇女不会因为没有理由地被抛弃, 或被置于社会和经济的弱势地位. 同样, 现代人在做离婚这个痛苦的决定前, 也必须想到, 耶稣首先关心的是信仰团体中, 每一个人, 作为天主的子女的幸福.

# 《偕主读经 -- 玛窦福音》

离婚的争论, 在有福音之前就已经存在. 耶稣时代, 关于这一争论出了两个极端的学派: 一个是特别随便的看法, 允许人可借任何理由离婚, 例如: 妻子烧焦或只是熏黑了饭, 丈夫就可以写休书将她休了. 所以, 他们按照这个派别的传统来解释法律, 一般允许离婚的条件, 是丈夫 "发现妻子有任何可的行为" ( 申 24: 1 ). 另外一个派别比较严格， 他们将《申命纪》中所说的例外情况仅仅解释为通奸行为.

法利赛人试图以离婚的问题来抓住耶稣的把柄. 耶稣却告诉他们一项积极的法律, 从天主起初所建立的原始秩序, 来说明这个问题, 不是强迫, 而是自由的选择 ( 创 1: 27， 2: 24, 5: 2 ).

在当时男人主导一切的社会里, 一个被抛弃的女人必须回到自己的父亲家, 这样使她自己, 以及整个家族蒙羞. 离婚的威胁如同残酷的武器, 迫使妻子屈服地丈夫. 在这种背景下, 耶稣的话带来完全的解脱和释放. 禁止离婚的目的主要是为了保护妇女, 并为恢复天主从起初建立的秩序.

希腊文 "淫乱" ( 9 ) 一词的另一种意思是 "姘居", 即男女不合法的结合. "不能结婚生育" 和 "放弃了婚姻" ( 12 ), 原文直译为 "阄人".

门徒们听到这种不可拆散的约束时, 都感到非常惊讶 ( 法利塞人已离场, 没有参与讨论 ). 耶稣并没有取消已经说过的话, 而是更进一步, 提出了一个将要出现在自己团体中的情况, 即自愿独身生活 ( 12 ). 基督徒的独身, 只有在天国奥秘的意义下才能理解. 为此, 耶稣也说: "谁能接受就接受吧".

# 耶稣祝福儿童

# 经文

13 那时, 有人给耶稣领来一些小孩子, 要他给他们覆手祈祷, 门徒却斥责他们. 14 耶稣说: "你们让小孩子来吧! 不要阻止他们到我跟前来, 因为天国正属于这样的人." 15 耶稣给他们覆了手, 就从那里走了.

# 《新约导读系列 -- 玛窦福音》

为了解释这个故事在宗徒传教之中出现, 有人认为初期教会藉此故事传达关于该童领先的教义. 那句 "不要阻止他们" 是最初洗礼里最裹引.

# 《新学院村圣经诠释 -- 玛窦福音》

现在玛窦圣史以广角镜头将被描述的对象集中在家庭单位中最弱势的成员身上 ( 13 ~ 15 节 ). 在这一段与先前的情节连结起来, 就可以看到耶稣对小孩子的祝福和祈祷, 是意识到父母打算离婚时, 最受伤害的中能是小孩. 门徒为什么要阻止孩子到耶稣跟前, 圣史并没有说明原因, 耶稣曾在之前的言论谈到过关于小孩在天国的重要性 ( 玛十八 ).

# 《偕主读经 -- 玛窦福音》

玛窦早已将儿童作为门徒们应效仿的模范. 耶稣的态度与门徒们的厌烦形成了稣明的对比. 耶稣借着这个象征性的行为强调: 天国永远优先属于那些将自己变为小孩的人, 或者那些以谦虚和纯朴的态度接受天主因赐的人.

# 财帛的为害

# 经文

13 有一个人来到耶稣跟前说: "师傅! 我该行什么 '善', 为得永生?" 17 耶稣对他说: "你为什么问我关于 '善'? 善的只有一个. 如果你愿意进入生命, 就该遵诫命." 18 他对耶稣说: "什么诫命?" 耶稣说: "就是: 不可杀人, 不可奸淫, 不可偷盗, 不可作假见证; 19 应该孝敬父母, 应该爱你的近人, 如爱你自己." 20 那少年人对耶稣说: "这一切我都遵守了, 还缺少什么?" 21 耶稣对他说: "你若愿意成全的, 去! 变卖你所有的, 施舍给穷人, 你必有宝藏在天上; 然后跟随我." 22 少年人听这话, 就忧闷的走了, 因为他拥有许多产业.

23 于是, 耶稣对门徒说: "我实在告诉你们: 富人难进天国. 24 我再告诉你们: 骆驼穿过针孔, 比富人进天国还容易. " 25 门徒们听了, 就非常惊异说: "这样, 谁还能得救呢?" 26 耶稣注视他们说: "为人这是不可能的; 但为天主, 一切都是可能的."

27 那时, 伯多禄开口对他说: "看, 我们舍弃一切, 跟随了你; 那么, 将来我们可得到什么呢?" 28 耶稣对他们说: "我实在告诉你们: 你们这些跟随我的人, 在重生的世代, 人子坐在自己光荣的宝座上时, 你们也要坐在十二宝座上, 审判以色列十二支派; 29 并且, 凡为我的名, 舍弃了房屋, 或兄弟, 或姊妺, 或父亲, 或母亲, 或儿女, 或田地的, 必要领取百倍的赏报, 并承受永生. 30 有许多在先的要成为在后的, 在后的要成为在先的. "

# 《新约导读系列 -- 玛窦福音》

耶稣遇到一个富人, 他谨法律, 无懈可击, 却不愿按受福音的神贫. 这给了他一个随机施教的机会，告诫宗徒们富有的人被允许入天国是何其困难. 那句 "是的, 我再说一次" ( 耶稣以前从来没有说过这种话 ) 是玛窦意图简略的表示. "针孔" 可能指一种很窄的城门, 也可能就是字面的意义, ( 是东方式的夸张的一例 ). 宗徒的反应 ( 对我们而言真是奇怪非常 ) 是以旧约时代的观点为根据, 绝不是人力所能影响的. 伯多禄询问关于他们十二人抛弃一切追随耶稣所应得的赏报, 耶稣答应给他们在将来都会的审判权 ( 旧约果所有与这件事情相关的语句切不可误会是指末世的审判 ). 那些放弃家庭和一切所有的基督徒都在这里得到此后和永生的 "许多倍" 的赏报. 这段文字里, 这一崇高的理想就成为众人知晓的福音劝谕 ( 即三愿 ).

# 《新学院村圣经诠释 -- 玛窦福音》

耶稣和富少年之间的对话, 以及随后和门徒的讨论, 都严肃地阐明财富可能成为门徒的障礙 ( 绊脚石 ). 玛窦圣史解释了 ( 参阅: 谷十 17 ~ 31; 路十八 18 ~ 30 ) 富少年问耶稣: 要如何行善, 才能承受永生? 这是玛窦圣史最喜欢描写的主题之一 ( 玛五 16: 七 17 ~ 19; 十二 12, 33 ~ 35 ; 十三 23, 24; 二十六 10 ). 尊守诫命是行善的第一步. 富少年问 "什么诫命?" 这个问题, 听起来不合适, 因为我们必须要遵守, 并同等对待所有的诫命.

耶稣不只邀请富少年遵守诫命而已, 还邀请他成为 "成全的" ( 参阅: 玛五 48 ), 这项邀情所关注的, 是要人成为 "满全的" 或 "完备的". 正如玛五 48 所示, 这项邀请并不是针对少数被拣选的人, 也不是要显现犹太教和基督信仰之间的差异性. 在旧约经书中, 虽然财富被认为是蒙受天主祝福的象征 ( 申二十八 1 ~ 14 ), 但亦是有关于财宝能腐蚀人心的可怕警告 ( 则七 19; 亚六 4 ~ 8; 葴十五 16 ).

由玛窦的观占看来, 作为耶稣的门徒, 就得对耶稣所诠释的犹太法律具有信德, 其中就包含了必须极端地信赖耶稣. 富人进入天国如同骆驼挤过针孔一样困难. ( 24 节 ). 很多人说 "针孔" 是耶路撒冷大城门上的一扇小门的名字 ( 骆驼并非穿不过, 但是很困难 ), 这是没有依据的. 对于门徒的惊讶 ( 同样情形发生在玛十九 10 ), 耶稣的回应是重新把门徒的注意力, 集中在天主对他们的主动邀情, 以及赐予他们的能力上, 这使门徒有能力行善, 这也就是富少年在一开始就提出的问题 ( 16 节 )

真福八端中有 "神贫的人是有福的" ( 玛五 3 ); 耶稣也说过: "你的财宝在那里, 你的心也必在那里" ( 玛六 21 ); 并警告说: 没有人 "能事奉天主而又事奉钱财" ( 玛六 21 ); 天主的国比要寻找的珍宝贵 ( 玛十三 44 ); 门徒得到天主赏报的主题, 贯穿整部《福音》( 玛五 12, 46 ~ 47; 六 1 ~ 6; 18; 十 39 ~ 42; 二十五 21, 23, 34 ). 这里的重点是末世, 因为门徒将分享天主的荣耀, 并参与人子在末世时的审判, 因为门徒为天主国度的自我奉献, 已使他们对领受天主为世人永远继承的旨意做了准备.

# 《偕主读经 -- 玛窦福音》

耶稣恢复了先知的一个原则: 财富是通向天国途中的巨大障碍. 可以说, 这一段是福音中最能影响基督宗教历史的精彩片段之一. 为那个富少年来说, 做耶稣的门徒所要付出的代价似乎太大, 因为他很富有 (22). 其实, 他期望耶稣告诉他另外一些事, 比如要求他做一些善事, 大量地施舍等等; 换句话说, 他希望在不影响自己现有生活方式的情况下，运用财富做些事情. 然而, 在天主面前, 奉献上所有的一切, 否则就等于没有奉献.

耶稣这些话, 已经激励了一些人以极端的方式跟随祂 ( 比如亚西西的圣方济各 将自己所有的财物者送给穷人), 教仁也由这些话引申出 "福音劝谕". 这种观谕有别于 "诫命", 并不是每个人的义务. 但是, 耶稣对于这个年轻人所说的话, 却不是劝谕, 而是个人的, 不可回避的命令 (21). 当财物变成我们跟随耶稣和寻求天国的障碍时, 弃绝财富已经不再是劝谕, 而是绝对履行的要求. 一个人不可以同时待奉天主, 而又待奉财富.

耶稣继续跟门徒们讲述财富阻碍进入天国 ( 23 - 26 ). 骆驼和针孔的图像描述是中东文化的一种夸张手法, 说明一个人, 要抛弃财富所带来的权力和荣誉是何等的艰难和不可能.

耶稣向那个富少年所提出的两个要求, 门徒已经做到了: 他们舍弃一切并跟随了祂. 为此, 他们的回报将是一个完满的生命, 而且已经预先尝到了这生命的奇妙. 耶稣的许诺也延伸至所有为了祂的缘故而舍弃一切的人, 具体地说就是所有信徒. 尽管无法确定每个人在天国的地位, 但是, 每个人所得的回报肯定远远大于自己所想象的 ( 30 ). 十二个光荣宝座的意思, 就是要求让徒们服待并培育那些天国的天主子民.
