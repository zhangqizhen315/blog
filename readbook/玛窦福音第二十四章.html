# 玛窦福音第二十四章

# 预言耶路撒冷和圣殿的毁灭

# 经文

1 耶稣由圣殿里出来前行，门徒前来把圣殿的建筑指给他看。2 耶稣回答他们说：“你们不是看见这一切吗？我实在告诉你们：将来这里决没有一块石头，留在另一块石头上，而不被拆毁的。”3 他在橄榄山上坐着的时候，门徒前来私下对他说：“请告诉我们：什么时候要发生这些事？又什么是你来临和时代穷尽的先兆？”4 耶稣回答说：“你们要谨慎，免得有人欺骗你们，5 因为将有许多人假冒我的名字来说：我就是默西亚；他们要欺骗许多人。6 你们要听到战争和战争的风声。小心，不要惊慌！因为这是必须发生的，但还不是结局，7 因为民族要起来攻击民族，国家要起来攻击国家；并且到处要有饥荒、瘟疫和地震。8 但是，这一切只是痛苦的开始。

9 那时，人要交付你们去受刑，要杀害你们；你们为了我的名字，要为各民族所憎恨。10 那时，必有许多人要跌倒，互相出卖，互相仇恨；11 许多假先知要起来，欺骗许多人；12 由于罪恶的增加，许多人的爱情必要冷淡。13 唯独坚持到底的，才可得救。14 天国的福音必先在全世界宣讲，给万民作证；然后结局才会来到。

15 所以，几时你们见到达尼尔所说的‘招致荒凉的可憎之物’已立于圣地，──读者应明白： 16 那时在犹太的，该逃往山中；17 在屋顶上的，不要下来，从自己的屋里，取什么东西；18 在田野里的，也不要回来取他的外衣。19 在那些日子内，怀孕的和哺乳的，是有祸的。20 你们当祈祷，不要叫你们的逃遁遇到冬天或安息日。”

# 《新约导读系列 -- 玛窦福音》

这番道理可分为三段, 主要是有关耶路撒冷的毁灭. 然后以三个寓言来说明教会里某群人物的命运, 以及一个对最后审判有描绘.

当耶稣宣佈圣殿毁来时, 门徒问他这事发生的 "时间" 和 "先兆". 只有玛窦保留了他们内心的疑惑, 并且反映在第二个问题上: "什么是你来临和时代穷尽的先兆?" 门徒们 ( 错误地 ) 假设圣殿的末日就是指世界的末日, 因为依撒意亚描述写的默西亚时代是以圣殿为中心. 其实这个灾变指的是旧约世界的毁灭, 以及基督信仰脱离犹太教而获得自由. 耶稣拒绝泄露这事 "何时" 发生, 但他的确预言了那个 "先兆", 并在公元七十年以前都实现了. 许多假先知会相继出现, 人们会听到战争的风声. 基督徒要提防不要为假先知所骗, 也不要被战争风声惊吓; 这 "还不是结局". 这种政治上和自然界的剧变是 "苦痛的开端", 结果造成教会时代的诞生. 在公元七十年以前的年代里, 年青的教会将遇到一连串的危机; 殉教, 背教, 最初所有史弟般热爱的消逝. 不过耐心的坚持将会获得最后的胜利. 福音将遍传整个地中海岸的国家, 保禄在公元五十八年就证实了这一点, 然后 "结局才会到来".

# 《新学院村圣经诠释 -- 玛窦福音》

耶稣和圣殿宗教领袖的关系日趋紧张. 在圣殿里, 耶稣首先以先知性的行动清除了圣殿 ( 玛二十一 12 ~ 17 ), 并与圣殿宗教领袖们进行辩论 ( 玛二十一 23 ~ 二十二 46 ), 还针对宗教领袖的伪善向门徒警告 ( 玛二十三 1 ~ 36 ). 接着, 耶稣预言圣殿将被毁 ( 玛二十四 1 ~ 2 ), 把耶肋米亚先知指出默西亚来临之时辰所要发生之事 ( 耶七 1 ~ 15 ) 联系在一起了. 在玛窦圣史编写福音的时代, 圣殿已经被毁掉了. 但, 在耶稣受犹太宗教领袖审询时, 做假见证的控告耶稣曾威胁要拆毁圣殿 ( 玛二十六 61 ), 并在耶稣被钉十字架之处, 路过的人也为此辱骂了耶稣 ( 玛二十七 40 ).

接着, 耶稣谈及了末世的征兆. 耶稣像有权威的老师那样 ( 参阅: 玛五 10: 十五 29 ), 从在橄榄山上; 橄榄山是末世最后审判之地 ( 匝十四 4 ). 门徒在此和耶稣私下教导他们比喻的意义一样 ( 玛十三 10 ~ 17 ), 也得到了耶稣私下的教诲. 耶稣为门徒描绘出一幅世界极度混乱, 大规模遭到毁坏的图像, 到处都是 "假默西亚", 战争, 饥荒, 地震, 迫害, 因耶稣的名受人憎恨, 罪孽, 背叛, 欺骗, 不遵守法律, 人失去热情, 以及来自内心及处在两方面的争斗冲突.

几乎每一年龄层的人, 在看到这些征兆时都会怀疑: 这是否预示着末世的到来. 耶稣不久前派遣门徒出外宣讲时, 也曾提及了类似的主题 ( 玛十 16 ~ 25, 34 ~ 39 ) 而向门徒保证: 面对任何事情的发生, 都不要害怕, 因为天主会给他们每时每刻的照顾 ( 玛十 26 ~ 33 ). 在这里, 同样地, 耶稣告诉门徒, "唯独坚持到底的, 才可得救" ( 13 节 ). 这些现象 "只是痛苦的开始" ( 8 节 ), 犹如产痛是新生命的前奏一样. 玛窦圣史指出结局 ( 末世 ) 来到时, 会有一个标记, 就是天国的福音要先传遍天下 ( 也参阅: 玛二十八 16 ~ 20 ).

# 《偕主读经 -- 玛窦福音》

《玛窦福音 》的第二十四和二十五章为一个单元, 是五篇重要讲道中的最后一篇, 有关末世的讲道. 它可以分为三部分: 一、描述将要发生的事件 (24:1 -44); 二、劝导人们保持警醒并祈祷 (24:45-25:30); 三、审判的比喻 (25: 31 - 46).

玛窦设想, 耶稣从圣殿出来, 在远处沉思圣殿将来的命运. 这种图像具有象征意义: 耶稣走出了圣殿, 这也是祂最后一次离开圣殿, 永远地离开. 随后, 祂回到自己的门徒当中, 即那个新的团体. 这个由大黑落德所建造, 有雄伟石柱和壮丽殿廊的圣殿, 在耶稣口中成为渡向末世主题的跳板.

这可能是《玛窦福音》中最为费解的一段, 因为所讲述的都是未来的事件. 事件中的各个细节, 我们无从知晓, 而且其写作角度也具有时空重叠性, 反映宗徒们似乎有意合并且混淆两件事情, 即圣殿被毁 ( 历史时空中发生的事 ) 和默西亚来临时的世界末日 ( 历史时空的终结 ). 他们在寻求一些具体的标记, 好能制造一个日历, 较为准确地计算未日来临的时间. 他们这样问, 既是出于好奇, 又出于恐惧.

耶稣的回答是拒绝肯定任何现世要发生的事. 祂的话都在勉励门徒们保持警醒, 好能面对某些必然的苦难. 门徒们询问两件事情, 可是他们并不确定这两件事情之间的关系. "这些事" 是指圣殿被毁 "祢的再来" 指主耶稣带着圆满的光荣来临的日子, 也就是世界终穷的时刻.

在终结的时候, 会有一连串的重大事情发生 ( 4 - 14), 但是, 这些事肯定不会按照人们所预期的时间顺序发生. 人们的内心将要混乱, 人与人之间充满怨恨与争斗、天灾人祸、迫害、屠戮等等, 这一切就如同新时期诞生之前必然要经历的产痛. 最终, 人必须忍受这些痛苦, 耐心地等待, 因为天主将使一切痛苦升华 (9), 福音将会传遍世界 ( 14 ), 忠信的人都必得救 ( 13 ).

世界末日不会立刻到来, 今日流行的看法也是如此. 末日来临之前要出现一些预兆作为标记, 但所列举的标记绝对不可以被理解为末日来临的时间表. 历史上, 人们曾多次预言末日来临的时刻, 但这些预言都被证明是错误的! 这种对末日的计算和预测, 都与福音的精神背道而驰, 因为, 所有这些讯息都在劝勉人们, 要警醒 ( 25:13 ), 善用 "塔冷通" 并结出相应的果实 ( 25:14 - 30 ), 同时还要帮助有需要的人 ( 25:31 - 46 ). 如果福音给我们指出未来, 那也是为了帮助我们更好地投身于现今的历史.

# 人子的来临

# 经文

21 因为那时必有大灾难，是从宇宙开始，直到如今从未有过的，将来也不会再有；22 并且那些时日如不缩短，凡有血肉的，都不会得救；但为了那些被选的，那些时日，必将缩短。23 那时，若有人对你们说：看，默西亚在这里！或说：在那里！你们不要相信，24 因为将有假默西亚和假先知兴起，行大奇迹和异迹，以致如果可能，连被选的人也要被欺骗。25 看，我预先告诉了你们。

26 为此，如果有人对你们说：看，他在旷野。你们不要出来。或说：看，在内室。你们也不要相信。27 因为如同闪电从东方发出，直照到西方，人子的来临也要这样。28 无论在那里有死尸，老鹰就聚集在那里。”

# 《新约导读系列 -- 玛窦福音》

犹太的基督徒一旦看到 "招致荒凉的可憎之物" ( 达尼尔用这话来描述公元前一六七年, 安提约古第四厄丕法乃在全燔祭坛上建立的希腊神宙斯 ), 就应当立刻逃离圣城, 玛窦在这里所描述的历史事件是公元六八年开始提托进犯圣城. 罗马人建立偶像的习惯在犹人无偶像的传统中显得可憎. 历史告诉我们, 耶路撒冷的群众接受了这个暗示, 后来避难到外约旦的贝拉.

玛窦还加了 "或安息日", 否则那些犹基督徒考虑到安息日的休息, 可能会妨碍他们的逃脱. 他引用了 "人子的再临" 这个名词来表示, 他认为耶路撒冷的毁灭, 是光荣的基督来访. parousia 这个字, 因为保禄的影响, 只用来指基督在世界末日的第二次来临. 可是玛窦在此处及门徒最初之问题 ( 二十四 3 ) 中运用此字是指广意的基督再临. 马尔谷和路加都避免这种用法, 因为他们是保禄的弟子. 那闪电的喻象可能指在殿灭时, 基督来临是完全可见的.

# 《偕主读经 -- 玛窦福音》

一段采用了默示文学的写作风格, 引用达尼尔先知暗示的公元前一六七年, 安提约古第四约圣发乃在全熔祭坛上建立希腊神宙斯像 ( 15; 参阅达 9:27 ), 强迫犹太人崇拜异教神明的历史事件(取自光启《新约导读丛书一一玛窦福音》), 具体地描述了耶路撒冷的毁灭, 但其主要目的在于鼓励门徒们从这些灾祸中学得教训, 不至于误信人子来临的谎言. 这 "一场浩劫" ( 21 ) 将挑战人的信德, 甚至被拣选的人在得救的事上也要面临危机. 但是, 天主出自爱的干预将会缩短这段时期 (22). 他们应该提防假先知和假默西亚 ( 23 - 28 ), 这些人甚至能将被拣选的人带入歧途. 重复地强调 "不要相信" ( 23, 26 ), 是要教训他们, 在这些灾难和骗子面前时常保持清醒.

至于宇宙间的混乱和最后的荒凉景象, 是为指出末日的审判将突然降临到每一个人头上.

# 人子来临的先兆

# 经文

29 “那些时日的灾难一过，立时太阳就要昏暗，月亮也不发光，星辰要从天上坠下，天上的万象也要动摇。30 那时，人子的记号要出现天上；地上所有的种族，都要哀号，要看见人子带着威能和大光荣，乘着天上的云彩降来。31 他要派遣他的天使，用发出洪声的号角，由四方，从天这边到天那边，聚集他所拣选的人。”

# 《新约导读系列 -- 玛窦福音》

此处用默示禄体的词句是为了描述耶路撒冷的毁灭, 而不是世界的末日. 旧约时的先知曾经用这种宇宙性的词句来形容厄东的毁灭, 法老的死亡, 犹大和耶路撒冷的被攻. 这种文学体裁不仅是诗歌的喻象, 更是旧约司祭神这的表达法. 因为雅威是历史事件的作者, 也是宇宙的创作者, 精神的和物质的世界之间, 本来存在着一种先存的和谐. 所以, 任何人事的危机 ( 如亚述人或巴比伦人进攻巴勒斯坦 ), 都会引起自然界一种同情的回响. 新约以为耶稣的死亡和复活造成了一个 "新天新地", 这个观念就是相信天主是历史的全能引领者的反映. "人子的记号在天上出现" 使普世各国皈依; 按照匝加利亚所预言的这 "记号", 就是指以 "初传" 来宣扬耶稣救赎性的死亡与复活的初期教会, 成为 "向列邦高举的记号", 其 "可见性" 原先被它与犹太教的关联弄得朦胧不清, 直到圣殿倾毁以后, 才得彰现. 玛窦描绘 "天使" 以号角来聚集新的 "选民", 可能是象征着在宗徒领导下的教会所表现的统一与普世性.

# 《新学院村圣经诠释 -- 玛窦福音》

当末世到来时, 将会出现确凿无误的征兆. 它会像越过天际的闪电 ( 27 节 ), 或老鹰在死尸上盘旋 ( 28 节 ) 那样的明显. 有一个征兆的出现， 会像达尼尔先知所说的 "招致荒凉的可憎之物" 出现在圣殿内 ( 15 节; 达九 27; 十一 31; 十二 11 ). 《达尼尔先知书》所说 "招致荒凉的可憎之物", 指的是主前 167 年安提约古四世厄丕法乃 ( Epiphanes ) 亵渎圣殿, 在圣殿内设立了异教的祭坛及偶像 ( 可憎之物 ), 导致了玛加伯的起义; 而玛窦团体记忆犹新的是在主历 40 年时, 当时的罗马皇帝卡利古拉 ( Caligula, 37 ~ 41 ) 曾威胁要采取相同的行动来亵渎圣殿.

将来如果在圣殿发生妄做各种亵渎之事时, 信徒就应明白, 大灾难近了. 面对这空前的大灾难, 只有尽快逃奔, 迟延不得 ( 如同: 玛二 12 ~ 13; 十 23 ). 怀孕的妇人及哺乳孩子母亲行动不便, 通常是最大的受害者. 门徒应祈求大灾难不要发生在使逃亡更加困难的日子, 如 "冬天或安息日" ( 20 节 ) . 在安息日逃亡, 可能会引起团体的注意, 从而使逃亡者处于危险的境地. 如果有人认为逃亡会违反遵守安息日, 这样就可能成为团体分裂的原因.

人子来临前, 星象的徵兆 ( 参阅: 依十三 10; 三十四 4; 则三十二 7; 岳二 10, 31; 三 4; 四 15; 亚八 9; 盖二 6, 21 ) 是末世先兆的开端 ( 参阅玛八 20 之诠释, 见本书 87 ~ 88 页 ). 不了解为什么 "哀号" ( 30 节 ) 会是伴随人子来临的先兆? 是因为忧患, 苦难? 还是因为人们在忏悔? 末世时, 天主要聚集祂所拣选的子民, 这是一个常见的主题 ( 申三十 3 ~ 4; 依十一 11 ~ 12; 则三十七 21 ; 三十九 27 ~ 29; 匝二 6 ~ 12 ).

# 《偕主读经 -- 玛窦福音》

有关默西亚最后来临的景象描述, 充满先知性和默示性的文学特征, 所用的语言极富象征意义, 是为说明子在世界末日, 以天主的名字所执行的最后审判. 当作者将前面所描述的事件, 穿插在现今的历史背景中时, 基督的再来就标志着历史的终结. 基督再来就是历史的目标、终结和决定性的事件, 也是整个人类历史的意义所在. 每一代人都经验到人子的审判, 而那些能够坚持到底的, 将会获得永远的救恩.

基督教会传统中的象征学都认为 "人子的记号" 就是十字架 (30).

# 无花果树的比喻

# 经文

32 “你们应该由无花果取个比喻：几时它的枝条已经发嫩，叶子出生了，你们就知道夏天近了。33 同样，你们几时看见这一切，你们就该知道：他已近了，就在门口。34 我实在告诉你们：非到这一切发生了，这一世代决不会过去。35 天地要过去，但是我的话，决不会过去。”

# 《新约导读系列 -- 玛窦福音》

这个简单的寓言解释了耶路撒冷的毁灭和教会内天国建立的关系. 无花果树和夏天同是默西亚真福的记号. 对有信德的人而言, 圣城倾毁的实际意义是 "他已经来临了", 就是说升天的基督藉着教会脱离犹太教的事实, 来证明他的力量. 耶稣重复说他所预言的事将要在这一世代的有生之年发生. 虽然旧世界要消逝, 旧约的神圣制度告一结束, 但是耶稣的启示 -- 取代旧约的 "新创造", 却是永久而确定的.

# 醒寤的紧要

# 经文

36 “至于那日子和那时刻，除父一个外，谁也不知道，连天上的天使都不知道。 37 就如在诺厄的时日怎样，人子的来临也要怎样。38 因为就如在洪水前的时日，人照常吃喝婚嫁，直到诺厄进入方舟的那一天，39 仍然没有觉察；直到洪水来了，把他们都卷了去；人子的来临，也必要这样。40 那时，两个人同在田间，一个被提去，一个却被遗弃；41 两个女人同在磨旁推磨，一个被提去，一个却被遗弃。

42 所以，你们要醒寤，因为你们不知道：在那一天你们的主人要来。43 这一点你们要明白：如果家主知道，盗贼几更天要来，他必要醒寤，不让自己的房屋被挖穿。44 为此，你们应该准备，因为你们不料想的时辰，人子就来了。”

# 《偕主读经 -- 玛窦福音》

没有人知道末日来临的时刻. 故意不让人知道那日期, 是为激发人们时时警醒, 不能忽视现在的时日. 无花果树的注解, 劝导人要懂得分辨. 这是门徒在末日时刻来临之前必须具备的态度.

对于这即刻的来临, 不可以从时间的意义上理解, 而应该从神学的意义上理解. 这里所强调的和后面比喻所说的, 都肯定上主会随时到来. 这种肯定是为激励团体, 不要在现世妄求一种永恒的国度.

当门徒们从询问事件转为询问时间时, 耶稣的回答非常明确: 没有人知道 ...... 只有天父知道. 虽然我们不能知道末日来临的日期和时间, 但却应该时常记住, 人子必将来临. 我们必须要警醒, 并作好充分准备.

面对众多没有把握的事件和一切似乎显得失败且令人沮丧时, 耶稣以一个绝不动摇的信念支持信徒们 "天地都会消逝,但我的话永不会消逝" (35).

# 仆人等待主人的比喻

# 经文

45 “究竟谁是那忠信聪明的仆人，主人派他管理自己的家仆，按时配给他们食粮呢？46 主人来时，看见他如此行事，那仆人才是有福的。47 我实在告诉你们：主人必要委派他管理自己的一切财产。48 如果那恶仆心里说：我主人必要迟延，49 于是开始拷打自己的同伴，甚或同醉汉一起吃喝。50 正在他不期待的日子，和想不到的时刻，那仆人的主人要来到， 51 铲除他，使他与假善人遭受同样的命运；在那里要有哀号和切齿。”

# 《新约导读系列 -- 玛窦福音》

耶稣无法告诉他的门徒们 "何时" 这事会发生, 这是天父所持有的机密 耶稣被遣来世上的任务并不包括启示这个机密. 但是, 如诺厄同时代的人一样, 世人已受到警告: 耶路撒冷的毁灭无可避免. 然而像诺厄时代, 大中部份的人民没有防范洪水的突袭, 只有遗民 "如诺厄家" 被保全. 耶稣强调那重要的教训: 要醒寤, 要随时准备好. 他把自己比擬成一个盗贼 ( 这个比擬, 若不是耶稣自己首创, 没有任何基督徒敢引用 ), 在新约其他部分, 这种隐喻用来表达他们对基督在末日来临的信心: 历史终结以前将不会有任何 "征兆", 使人们得以预知时日. 监督家仆者的比喻, 是一个教导宗徒们宗徒们对他们的羊群分层负责的教训 ( 保禄显然是如此来了解这个比喻 ). 天国并非一个安宁的 "退休", 而是更积极的效忠基督, 参与推广天国的工作.

# 《新学院村圣经诠释 -- 玛窦福音》

在此, 一系列的比喻和象征性的言论, 都是在警告门徒要醒寤. 无花果树 ( 32 ~ 35 节 ), 与在巴勒斯坦其它大多是常青的树种不同, 无花果树会在冬天掉落全部的树叶. 正如它的发芽是夏日临近的记号, 前面几节的记述, 也是在提醒门徒人子的来临. 这里末世即将来临的郑重宣告 ( 34 节: "非到这一切发生了, 这一代绝不会过去" ), 与前述 "天国的福音必先在全世界宣讲, 给万民作证, 然后结局才会来到" ( 14 节 ) 的断言, 这两者论点之间, 存在着不协调. 门徒要做好准备, 并坚定不移地相信耶稣的话, 祂的话绝不会过去 ( 与旧约梅瑟法律相同, 参阅: 玛二十六 38, 40, 41 ).

前面数节强调的, 是在人子到来时, 人要保持醒寤; 而信的仆人这个比喻 ( 45 ~ 51 节 ), 则是劝诫门徒在主回来前必须对每日所行的式作保持警醒. 仆人的任务之一, 是分配日用的食粮 ( 45 节 ). 这个细节也许是暗指初期教会在食物分配和膳食方面所遇到的困难, 例如: 外邦基督徒和犹太基督徒一起进餐时 ( 迦二 11 ~ 14 ), 或不同社会地位的人在同一张餐桌上进餐时 ( 玛二十二 1 ~ 14 ), 所产生的冲突.

另外, 分配的食物可理解为 "天主的教导", 这是玛窦圣史的隐喻笔法 ( 如: 格前三 2; 若六 25 ~ 33 ), 而忠信仆人的比喻, 也是对团体领导者行使教导使命的劝勉. 对侵占团体资源的人, 耶稣对他们的警告相当可怕; 同时, 受遣责的人也要被铲除 ( 51 节, 直译为 "分割为二" ).

# 《偕主读经 -- 玛窦福音》

仆人的比喻强调门徒们应有的警醒和恒心到底的态度. 仆人在主人归来时要得到报应, 或得到奖赏, 或受到惩罚.

这个比喻也是在嘱咐基督徒团体的领导们, 要有保持警醒的心态. 主人对负责任的仆人的要求也包括明智和警醒. 比喻也强调忠信的仆人将会获得天国的赏报, 而这个赏报就是天主自己和祂的事业, 不是其他任何事物. 这个赏报, 在以强权和法律主义所建立的社会里面是难以理解的.
